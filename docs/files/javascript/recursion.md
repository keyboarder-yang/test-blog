#  递归

## 定义

程序调用自身的编程技巧成为递归（recursion）。

## 思想

重复利用函数本身能实现的功能，按照顺序进行不断的自调用

## 核心

建立递归函数

+ 边界条件（递归结束条件）
+ 递归前进段
+ 递归返回段

当边界条件不满足时，递归前进；当递归条件满足时，递归返回。
