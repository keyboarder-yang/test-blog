#### 小知识

> 1. 类中必须有一个constructor构造方法，不写默认会被添加一个空的构造方法
>
> 2. 类中constructor构造方法内，this指向 new出来的实例。
> 3. 类是不存在变量提升的。
> 4. class中的取值函数get和存值函数set：类似与object.defineProperty中的get和set。可以实现对属性的拦截
